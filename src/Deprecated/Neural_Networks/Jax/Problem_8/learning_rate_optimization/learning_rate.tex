\hypertarget{boundary-conditions}{%
\subsubsection{Boundary conditions}\label{boundary-conditions}}

\(\psi(0,y)=\psi(1,y)=\psi(x,0)=0\) and
\(\frac{\partial \psi}{\partial y}(x,1)=2\sin(\pi x)\)

\hypertarget{loss-function}{%
\subsubsection{Loss function}\label{loss-function}}

The loss to minimize here is
\(\mathcal{L} = ||\Delta \psi(x,y) +\psi(x,y)\cdot\frac{\partial \psi(x,y)}{\partial y}-f(x,y) ||_2\)

\hypertarget{analytical-solution}{%
\subsubsection{Analytical solution}\label{analytical-solution}}

The true function \(\psi\) should be \(\psi(x, y)=y^2sin(\pi x)\)\\
This solution is the same of the problem 7

\hypertarget{approximated-solution}{%
\subsubsection{Approximated solution}\label{approximated-solution}}

We want find a solution \(\psi(x,y)=A(x,y)+F(x,y)N(x,y)\) s.t:\\
\(F(x,y)=\sin(x-1)\sin(y-1)\sin(x)\sin(y)\) \(A(x,y)=y\sin(\pi x)\)
